# Security Policy

## Supported Versions

This project is a personal learning/demo repository. Security fixes are provided on a best-effort basis.

## Reporting a Vulnerability

If you believe you found a security issue:

- Do not open a public issue with sensitive details.
- Email the maintainer with steps to reproduce and impact.

## Secrets

- Do not commit `.env` files, tokens, or keys.
- Use environment variables for all credentials.
